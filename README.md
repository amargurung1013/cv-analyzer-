# CV Analyzer with Supabase

This project is a FastAPI resume analyzer. A user uploads a PDF resume, the app extracts the resume text, sends it to the AI reviewer, shows the result page, and saves the analysis into a Supabase table.

The Supabase table stores:

- Professional Summary
- Strengths
- Weaknesses
- ATS Analysis
- Suggestions
- Career Level

## 1. Create a Supabase Project

1. Go to `https://supabase.com`.
2. Create a new project.
3. Open the project dashboard.
4. Go to `Project Settings`.
5. Open `API`.
6. Copy these values:
   - `Project URL`
   - `service_role` key

Keep the `service_role` key private. It should only be used on the backend in your `.env` file.

## 2. Create the Database Table

In Supabase, open `SQL Editor`, create a new query, and run this:

```sql
create table public.resume_analyses (
    id bigint generated by default as identity primary key,
    created_at timestamp with time zone default now() not null,
    filename text,
    professional_summary text,
    strengths text,
    weaknesses text,
    ats_analysis text,
    suggestions text,
    career_level text
);
```

After running the query, open `Table Editor` and select `resume_analyses`. This is where every analyzed resume result will appear.

## 3. Create the Environment File

In the project folder, create a file named `.env`:

```env
GROQ_API_KEY=your_groq_api_key_here
SUPABASE_URL=your_supabase_project_url_here
SUPABASE_SERVICE_ROLE_KEY=your_supabase_service_role_key_here
```

Example:

```env
GROQ_API_KEY=gsk_xxxxxxxxxxxxxxxxxxxxx
SUPABASE_URL=https://your-project-id.supabase.co
SUPABASE_SERVICE_ROLE_KEY=eyJxxxxxxxxxxxxxxxxxxxxx
```

## 4. Install Python Dependencies with uv

Install `uv` if you do not already have it:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Create a virtual environment:

```bash
uv venv
```

Activate the virtual environment:

```bash
source .venv/bin/activate
```

On Windows:

```bash
.venv\Scripts\activate
```

Install the dependencies:

```bash
uv pip install -r requirements.txt
```

## 5. Run the Website

Start the FastAPI server:

```bash
uv run uvicorn app:app --reload
```

Open this URL in your browser:

```text
http://127.0.0.1:8000
```

Upload a PDF resume and click `Analyze Resume`.

## 6. Check Supabase

After a resume is analyzed:

1. Go back to your Supabase project.
2. Open `Table Editor`.
3. Select the `resume_analyses` table.
4. You should see a new row with the filename, professional summary, strengths, weaknesses, ATS analysis, suggestions, and career level.

## How the Connection Works

The database code is in `database.py`.

```python
supabase = create_client(SUPABASE_URL, SUPABASE_SERVICE_ROLE_KEY)
supabase.table("resume_analyses").insert(payload).execute()
```

The save happens in `app.py` after the AI response is split into sections:

```python
save_resume_analysis(resume.filename, sections)
```

That means every successful analysis is saved automatically before the result page is shown.
